Описание: Корзина формируется классом RenderCart. Карточки товаров классом RenderCart. Данные поступают из массива goods (объект). Верстка карточек bootstrap v. 5 + свои анимации.

Функционал: Добавление в корзину посредством кнопки «Добавить в корзину». В корзине подсчитывается общее количество товаров. Очистка корзины. В корзине есть подсчет общей суммы каждого товара.
